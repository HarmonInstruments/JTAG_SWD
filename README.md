# JTAG_SWD

![photo of PCB](photo.jpg)

## JTAG

Hirose DF3A-6P-2DS header

## Adapter Cables

###

Hirose DF3-6EP-2C 6 pin inline header

TC2030-MCP
TC2030-MCP-NL

### Pinout

|Pin |JTAG | Serial Wire |
|----|-----|-------------|
| 1  | VCC | VCC         |
| 2  | TCK | SWCLK       |
| 3  | GND | GND         |
| 4  | TDO | SWDIO       |
| 5  | NC  | jumper to 6 |
| 6  | NC  | jumper to 5 |
| 7  | TDI | SWDIO       |
| 8  | TMS | NC          |
